# smooth-scroll
Vanilla JS smooth scroll by CSSLICK

## Usage

```
  <section id="page1">
    <h1>page 1</h1>
    <a class="btn_page" href="#page2">page2</a>
  </section>
  <section id="page2">
    <h1>page 2</h1>
    <a class="btn_page" href="#page3">page2</a>
  </section>
  <section id="page3">
    <h1>page 3</h1>
    <a class="btn_page" href="#page1">page2</a>
  </section>
```
